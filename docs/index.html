🪟 Windows 설치

  1. BirdPocket-1.0.3.exe 다운로드 → 더블클릭
  2. "Windows의 PC를 보호했습니다" 창 → 추가 정보 → 실행
  3. 🐦 화면에서 Google 로그인 → 시작!

🍎 macOS 설치 (Apple Silicon)

  1. .dmg 다운로드 → 열기 → 앱을 Applications로 드래그
  2. 처음 실행 시 "열 수 없습니다" → 완료
  3. 시스템 설정 → 개인정보 보호 및 보안 → 아래로 스크롤 → "그래도 열기"
  4. 비밀번호 / Touch ID → 열기
⚠️ 서명/공증을 하지 않은 개인 배포라, 처음 한 번 위와 같은 보안 허용이 필요해요(정상입니다). 인터넷은 로그인·저장에만 쓰고, 게임·사진·음악은 앱에 모두 내장돼 있습니다. · Intel Mac은 별도 빌드가 필요합니다.
